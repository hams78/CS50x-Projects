## 📂 Featured Projects in this Repository

### 💳 1. CS50 Finance
A full-stack web application that allows users to manage a mock portfolio of stocks.
* **Features:** Users can register/login, check real-time stock quotes, buy and sell shares (simulated with virtual cash), and view a complete history of transactions.
* **Tech Used:** Python, Flask, SQL (SQLite), HTML, CSS, Bootstrap.

### 🗳️ 2. Plurality
A command-line program written in C that simulates a plurality election.
* **Features:** Implements a voting system where candidates are ranked, votes are tallied, and the winner(s) are determined algorithmically.
* **Tech Used:** C (Data structures, linear search).

### 🚀 3. Final Project (`project.py`)
My custom capstone project for the CS50x course, showcasing independent problem-solving and implementation of Python programming principles.
